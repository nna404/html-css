Teste de titulo

Podemos simplificar esse codigo abaixo todo 

p{
        font-family: Arial, Helvetica, sans-serif;
        font-size: 1em;
        font-style: italic;
        font-weight: bold;
    }

Simplesmente usando:

p{
        font: italic bold 1em Arial, Helvetica, sans-serif;
    }

Devemos sempre lembrar da ordem dos atributos:

  • font-style
  • font-variant
  • font-weight
  • font-size/line-height
  • font-family
